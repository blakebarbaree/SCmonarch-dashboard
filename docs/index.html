Santa Cruz Monarch Movement Dashboard

This dashboard provides an interactive summary of Blu+ telemetry detections, movement episodes, and overwintering site use for western monarchs radio tagged in Santa Cruz, California, USA.

The dashboard is designed to help visualize detection histories, movement paths, site use, visit and foray episodes, and broad behavioral classifications through time and space.

How to Use This Dashboard

Use the navigation bar at the top of the page to move between the dashboard views:

  • all detections,
  • detections by individual tag,
  • all movement episodes,
  • episodes by individual tag.

Detection Explorer

The detection explorer shows detection-level movement records. It includes:

  • site boundary polygons,
  • detection points,
  • detection-step lines,
  • behavior classifications,
  • episode-type point symbols,
  • dashed transition lines between new episodes or dates.

Detections by Tag

The tag-specific detection pages show detection records for one monarch tag at a time. These pages are useful for reviewing the movement history of individual tagged monarchs.

Episode Explorer

The episode explorer summarizes movement episodes, including visits, forays, episode start and end points, and transitions among sites.

Episodes by Tag

The tag-specific episode pages show episode histories for individual monarchs. These pages are useful for reviewing site use, visit behavior, forays, and movement sequences by individual tag.
